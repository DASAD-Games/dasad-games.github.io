DASAD Games:
DASAD Games é um time de desenvolvimento de games criado entre alunos da Fatec Carapicuiba, com intuito de criar um jogo para o projeto SMAUG, e nesse semestre o game em questão foi Lury!


Smaug:
Você deve estar se perguntando "Mas o que é SMAUG?" SMAUG é o Sistema Maximizado de Avaliação Unificada em Games! Ou seja, é um projeto semestral onde os alunos do curso Tecnólogo em Jogos Digitais devem desenvolver um game.


Lury:
Vamos ao que interressa, Lury! O nosso game desse semestre! Lury é um game de plataforma, uma aventura onde o jogador controla a Lury, uma cachorrinha muito valente, em sua quest! Mas quem é Lury???


A História:
A história começa quando Lury e seus humildes donos são obrigados a migrar para outra região, pois o local onde viviam foi castigado por uma terrível seca.


Após uma longa caminhada, a família chega a uma misteriosa floresta e nela se depara com um grupo de criminosos que estão derrubando as árvores e capturando animais para traficá-los. A família é feita prisioneira pelos malfeitores.


Lury tenta proteger seus donos e morde o cruel líder dos criminosos. O impiedoso homem a chuta com demasiada força, ela bate a cabeça contra uma pedra e fica atordoada... Ela vê num borrão o bandido apontando sua arma para ela sem piedade... Tudo escurece, ela esta perdendo a consciência, não vê mais nada... Mas ouve o ensurdecedor disparo...
E agora??? O que aconteceu??? Só jogando pra saber...
